# India SaaS Starter

CodeForge template pack for B2B SaaS targeting the Indian market.

## Includes

- Supabase Auth (email + Google)
- Razorpay subscription billing (INR)
- Admin dashboard shell (Next.js)
- Deploy checklist for Vercel + Razorpay webhooks

## Usage with CodeForge

In chat, pick **India SaaS starter** or say:

> Use the `templates/india-saas-starter` scaffold. Copy files into my workspace, then customize for my product.

See `SCAFFOLD.md` for file layout and env vars.
