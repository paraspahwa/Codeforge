# CodeForge Product Roadmap

**Last updated:** June 2026  
**Primary wedge:** Affordable AI product partner for Indian founders and developers — idea → PRD → code → deploy in one workspace.

This document is the **product and GTM roadmap** (what we build for users and why). Engineering phase tickets live in [implementation-plan.md](implementation-plan.md).

**Public status:** `/roadmap` and `apps/web/lib/product-roadmap.js` (synced with shipped features).

---

## Roadmap phases (summary)

| Phase | Status | Highlights |
|-------|--------|------------|
| **A — Clarity & trust** | Shipped | Landing, `/roadmap`, quality metrics, deploy checklist, founder templates |
| **B — Founder wedge** | Shipped | Build journey tracking, India scaffolds, Hinglish, case studies, live preview |
| **C — Developer wedge** | Shipped | IDE parity, inline edit, AI Tab completion, parallel sessions UI |
| **D — Team & cowork** | Shipped | Teams, delegation, scheduled jobs + web notifications, SSO/audit, Zoho/Tally MCP |
| **E — Scale** | In progress | Voice, Figma design-to-code, mobile PWA, plugin marketplace; Mumbai hosting operator track |
